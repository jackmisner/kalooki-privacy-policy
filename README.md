# Privacy policy for Kalooki app

  - Kalooki does not collect, transmit, or share personal data
  - All game data (saves, statistics, settings) is stored locally on device
  - If the app crashes, you can choose to send a crash report (your device model, OS version, and the technical error details) to
  the developer by email — it is only sent if you explicitly choose to
  - The optional in-game "Report a problem" feature (solo games) lets you send a bug report — the current game state, the moves
  played that game, and your app version, device model, and display settings — only when you explicitly choose to. It is sent
  from your own email app; player names are replaced with generic placeholders (Player 1, Player 2, etc.), no account information
  is included, and your email address is removed before the developer receives the report
  - Local multiplayer uses Google Nearby Connections for peer-to-peer communication between nearby devices — no data is sent to
  external servers
  - Player names are user-chosen and not linked to real identity
